# <p align="center">How to Sart?</p>

- Create an ".env" file on your proyect root. It should look like this:

	    node_modules/
	    src/
	    .env <--------------- Your custom file

- Inside the ".env" file put your db login credentials

        user = <username>
        password = <password>
        dbName = <dbName>
